A premium, responsive **E-Commerce Frontend Application** built   using **Next.js 14 App Router** and **Tailwind CSS**.

Designed to deliver a modern shopping experience with fast performance, responsive layouts, intuitive product browsing, and seamless cart management.

## ⚙️ Installation Guide

### 1. Clone Repository

```bash
git clone https://github.com/Ayushpandey2026/whatbytes
```

### 2. Move into Project

```bash
cd whatbytes
```

### 3. Install Dependencies

```bash
npm install
```

### 4. Run Development Server

```bash
npm run dev
```

Open browser:

```bash
http://localhost:3000
```

## 🔥 Optimization Highlights

### Performance Improvements
✅ Implemented **Search Debouncing** to reduce unnecessary re-renders and improve filtering performance.

✅ Used **Next.js Image Optimization** for better image loading.

✅ Applied **useMemo optimization** for filtered product rendering.

---

### Stability Improvements
✅ Built using **Node.js v22 LTS** for stable modern runtime support.

---

### Architecture Improvements
✅ Context API used for centralized cart state management.

✅ localStorage persistence to retain cart across browser refreshes.

✅ Dynamic routing with Next.js App Router.
